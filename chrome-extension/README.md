# FAB戦績一覧表示 Chrome拡張機能

イベント履歴ページから戦績一覧を抽出して表示するChrome拡張機能です。

## インストール方法

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. この `chrome-extension` フォルダを選択

## 使い方

1. [GEMのイベント履歴ページ](https://gem.fabtcg.com/profile/history/)を開く
2. 拡張機能のアイコンをクリック
3. 「戦績を抽出」ボタンをクリック
4. 戦績一覧が表示されます
5. 「結果をコピー」ボタンで結果をクリップボードにコピーできます

## 出力形式

```
12/28 FABLE INVITATION CUP Fable 3-3
12/27 「Calling: Hong Kong」お土産争奪CC Fable 4-2
12/26 金曜CC Fable 0-3
```
